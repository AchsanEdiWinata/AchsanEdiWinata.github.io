Hobi

Berbicara tentang hobi, saya sangat suka dengan elektronika, saya sangat senang desain PCB, membuatkan jalur skematik, sampai memprogram. Saya mendapatkan hobi ini disaat saya kelas 3 SMK, jujur saja dulu saya sangat bosan dengan elektronika, karena hanya materi saja praktiknya jarang. Lambat laun disaat praktik pemprograman saya sangat antusias memperhatikan, yang saya tangkap tentang pemprograman adalah kita tidak boleh menghafal syntax atau error karena menurut saya akan susah jika menghafal syntax dan error, kita hanya harus memahami saja.
